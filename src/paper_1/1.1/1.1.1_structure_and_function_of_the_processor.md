# 1.1.1 Structure and function of the processor

## Registers
Inside of the CPU there are several registers which hold data that is currently in use by the CPU. 
The size of each register varies based on what type of register and the architecture of the CPU. For example, the floating point registers (XMM) on a typical x86_64 cpu are 128 bits large, however the typical registers on this CPU architecture are only 64 bits wide.


| Register                     | Abbreviation | Function                                                                                                           |
| ---------------------------- | ------------ | ------------------------------------------------------------------------------------------------------------------ |
| Program counter              | PC           | Contains the address of the next instruction to be executed from memory.                                           |
| Memory Address Register      | MAR          | Contains an address in memory that will be written/read from                                                       |
| Memory Data Register         | MDR          | Contains the data read from the address stored in the MAR, or data that is to be written to the address in the MDR |
| Current Instruction Register | CIR          | Contains the instruction currently being executed                                                                  |
| Accumulator                  | ACC          | Contains the output from the ALU, either logic or arithmetic                                                       |
